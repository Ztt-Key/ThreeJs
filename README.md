
这是一个使用vue3+threejs的新手入门练习项目

全部内容为个人喜好所做，所以没什么太大关联性，但可以作为一个很好的入门案例学习

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
